## Code Review

[Watch Code Review Video](https://drive.google.com/file/d/1bLQP5PmDJ1lEpLq6-7imdjk0ZF5-NiXn/view?usp=sharing)

This artifact is a code review of the Grazioso Salvare Animal Shelter dashboard developed in CS 340. The purpose of this review is to evaluate the original implementation, identify areas for improvement, and highlight enhancements related to software design, algorithms, and data handling.

### Summary
This project demonstrates the use of Python, Dash, and MongoDB to create an interactive dashboard for visualizing animal shelter data. The application allows users to filter data and view results through tables, charts, and maps.

### Code Analysis
The original implementation functioned correctly but included areas for improvement, such as inefficient filtering logic, lack of modular design, and security concerns related to hardcoded credentials.

### Planned Enhancements
Enhancements focused on improving code structure, optimizing filtering logic, and strengthening security practices. These improvements were designed to increase efficiency, maintainability, and overall system reliability.

### Skills Demonstrated
- Software design and modular development  
- Data processing and filtering logic  
- Database integration with MongoDB  
- Problem-solving and system improvement

## Database Enhancement

This enhancement focused on improving database performance by implementing MongoDB query-based filtering instead of processing data entirely in memory.

### Enhancements Made
- Implemented MongoDB query filtering using `$and`, `$regex`, and range operators  
- Reduced full dataset loading by querying only relevant records  
- Improved data handling and error management  
- Strengthened overall system performance
